# Doc Generator

Generate comprehensive technical documentation from codebases.

## Agents

| Agent | Purpose |
|-------|---------|
| api-documenter | Generate API reference documentation |
| docs-architect | Design documentation structure |
| mermaid-expert | Create architecture and flow diagrams |
| reference-builder | Build reference documentation |
| tutorial-engineer | Write step-by-step tutorials |

## Commands

| Command | Purpose |
|---------|---------|
| `/doc-generate` | Generate documentation for a codebase |

## Usage

```
/doc-generate
```

The command analyzes your codebase and coordinates specialized agents to produce:

- API documentation
- Architecture diagrams (Mermaid)
- Getting started guides
- Reference documentation
- Tutorials
